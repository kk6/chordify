# 🎸 Chordify

**Chordify** は、ギターコードを視覚的に記録・管理し、毎日のコード練習を効率的に支援するためのWebアプリケーションです。

🚀 開発タスク管理は [GitHub Projects運用ガイド](docs/07_project-management.md) に準拠しています。

## ✨ 主な機能

- ギターコードを視覚的に簡単登録・管理
- タグによるコード分類と検索
- コード響きを音声再生で確認
- 未習得コードを毎日ランダム表示
- 音名表示と度数表示をワンタッチで切り替え可能
- 学習進捗をグラフ化して表示
